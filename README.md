# Valet-sh Php

## How do I install these formulae?

`brew install valet-sh/php/<formula>`

Or `brew tap valet-sh/php` and then `brew install <formula>`.

Or, in a `brew bundle` `Brewfile`:

```ruby
tap "valet-sh/php"
brew "<formula>"
```

## Documentation

`brew help`, `man brew` or check [Homebrew's documentation](https://docs.brew.sh).
